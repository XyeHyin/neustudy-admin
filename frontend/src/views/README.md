## Views

Pages are grouped by feature area. Keep route-level pages here and place reusable UI in `src/components`.

- `activity`: activity log pages.
- `auth`: login, registration, and password recovery pages.
- `dashboard`: dashboard and landing-after-login pages.
- `errors`: forbidden and not-found pages.
- `learning`: categories, courses, and knowledge-point pages.
- `practice`: papers, practice sessions, grading, and practice statistics.
- `question`: question bank and AI question generation pages.
- `system`: user, role, permission, and profile pages.
