# Neustudy Admin Frontend

Neustudy Admin 的前端管理端，基于 Vue 3、TypeScript、Vite、Pinia、Vue Router 和 Naive UI 构建。

## Scripts

```bash
pnpm install
pnpm dev
pnpm lint
pnpm type-check
pnpm build
```

## Structure

- `src/api`: API request modules.
- `src/api/types`: API DTO/VO type declarations grouped by business domain.
- `src/components`: reusable UI components grouped by feature.
- `src/composables`: reusable Vue composition logic.
- `src/constants`: shared static options and constants.
- `src/router`: route definitions and route guards.
- `src/store`: Pinia stores.
- `src/utils`: shared utility functions.
- `src/views`: route pages grouped by business domain.
